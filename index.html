Plataforma de Gestão
Acesse com sua conta Google para sincronizar seus dados com o Google Drive.
Seus dados ficam apenas no seu Google Drive pessoal. Nenhuma informação é enviada a servidores externos.

⚙️ Configuração inicial

Para salvar seus dados, a plataforma precisa de uma pasta no seu Google Drive. Siga os passos abaixo:

  1. 1No Google Cloud Console, crie um projeto e ative a Google Drive API.
  2. 2Crie credenciais OAuth 2.0 (tipo: "Aplicativo da Web") com o domínio do GitHub Pages como origem autorizada.
  3. 3Copie o Client ID gerado e cole abaixo.
  4. 4A plataforma criará automaticamente a pasta RB-Advocacia-Dados no seu Drive.
Encontrado em: Google Cloud Console → Credenciais → OAuth 2.0
